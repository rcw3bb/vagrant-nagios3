### Install the Following

- vagrant
- virtualbox

### To start, execute the following batch:

```
nagios-start.bat
```
### To stop, execute the following batch:

```
nagios-stop.bat
```
### Accessing Nagios

Use the following URL:

> http://localhost:7777/nagios3

**Credentials**

| Username    | Password     |
| ----------- | ------------ |
| nagiosadmin | admin12345%# |
